Adjunto-o-Matic

Una herramienta para generar variable de adjuntos para Marketing Cloud.

1

Configuración

Primero, asegúrate de subir tu archivo a Marketing Cloud y luego completa este formulario.

Asegúrate que sea el nombre exacto y con extensión incluída. Debes copiar la clave completa.
2

Código

Esta es la versión para copiar y pegar directamente por ejemplo, en el editor de Marketing Cloud.

Hecho con ♥ por @notdarriagada

El nombre de archivo debe ser tal cual está en el repositorio de MKTCloud. Por compatabilidad, asegúrate de evitar tildes, carácteres especiales o doble espacio.

Esta variable es el identificador único de archivo, así se evitan problemas con archivos similares. La clave la encuentras una vez que subas el archivo. Aparecerá en la columna Clave del cliente, y al lado habrá un botón que dice Copiar.
